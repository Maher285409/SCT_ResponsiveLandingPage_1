# SCT_WD_1

## 🌐 Responsive Landing Page

This project is a responsive landing page with an interactive navigation menu. The navigation bar changes its style when the user scrolls or hovers over menu items, providing a dynamic and modern user experience.

---

## 🚀 Features

* Responsive design (works on mobile, tablet, desktop)
* Fixed navigation bar (always visible)
* Navbar style changes on scroll
* Hover effects on menu items
* Smooth and clean UI

---

## 🧠 How It Works

* The navigation bar is fixed at the top of the page
* When the user scrolls, the navbar changes color/style
* When hovering over menu items, visual effects are applied
* The layout adjusts automatically for different screen sizes

---

## 🛠️ Technologies Used

* HTML (structure of the page)
* CSS (styling and responsiveness)
* JavaScript (scroll and hover effects)

---

## 📌 Project Goal

The goal of this project is to understand responsive web design, navigation behavior, and basic DOM manipulation using JavaScript.
